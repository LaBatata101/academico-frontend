# Acadêmico

## Dependencias

-   `npm`
-   `react-scripts`

Para instalar o `react-scripts`, usar o comando `npm i react-scripts`

## Executando

```bash
$ git clone https://github.com/LaBatata101/academico-frontend
$ cd academico-frontend
$ npm start
```

Depois só ir em [http://localhost:3000](http://localhost:3000)
